## 🌟 Overview

The **Premium Student Grade Calculator** is a comprehensive solution for managing and calculating student grades. This repository contains two distinct, powerful interfaces that require no complex setups:
1. **Java Desktop Application**: A sophisticated, glassmorphic UI built with Java Swing for robust, compiled local calculations.
2. **Web Interface (Static Site)**: A beautiful, animated, and responsive frontend built with HTML/CSS and JavaScript—ready to run instantly in any web browser.

---

## ✨ Key Features

### 🖥️ Desktop App (Java Swing)
- **Glassmorphism UI**: Beautiful, translucent design with modern gradient backgrounds.
- **Real-time Validations**: Instantly validates marks to ensure they fall between 0 and 100.
- **Dynamic Color Coding**: Final grades are color-coded (Green for 'A', Red for 'F', etc.).
- **Interactive Elements**: Hover effects and fluid animations on buttons and input fields.

### 🌐 Web Interface (Vanilla JS)
- **Zero Configuration**: No servers required. Simply open `index.html`.
- **Fluid Animations**: Animated background blobs and glassmorphic form containers.
- **User Authentication UI**: Login and Registration mockups with smooth toggle transitions.
- **Responsive Design**: Flawless experience across desktop, tablet, and mobile devices.

## 🚀 Getting Started

### 1️⃣ Running the Web Interface

It couldn't be simpler! There's no server or dependency installation required.
1. Download or clone this repository.
2. Locate the `index.html` file in the main folder.
3. **Double-click `index.html`** to open it in your default web browser (Chrome, Firefox, Safari, Edge, etc.).

### 2️⃣ Running the Java Desktop Application

1. Ensure you have the **Java Development Kit (JDK)** installed on your machine.
2. Open a terminal or command prompt and navigate to the project directory.
3. Compile the Java files:
   ```bash
   javac GradeCalculatorUI.java GradeCalculator.java
   ```
4. Run the compiled application:
   ```bash
   java GradeCalculatorUI
   ```
